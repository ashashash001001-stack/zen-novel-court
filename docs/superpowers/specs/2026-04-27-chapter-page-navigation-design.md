# 章節頁導航功能設計

## 概述

在章節閱讀頁新增「更多」導航選單，讓用戶可以快速跳轉到首頁、書庫、我的頁面或當前書籍詳情頁。

## 設計決策

### 1. 位置
- **選擇：底部工具列**
- **理由：** 手機用戶單手操作時拇指更容易觸及，且與現有「目錄、設置、朗讀」工具按鈑風格一致

### 2. 呈現方式
- **選擇：選單式下拉面板**
- **理由：** 節省空間，避免與其他高頻按鈑混淆；這是現代閱讀應用的標準模式

### 3. 圖標選擇
- 使用「三條橫線」圖標 (☰) 表示「更多/選單」
- 與現有圖標風格一致（線框樣式）

## UI/UX 詳細設計

### 按鈑外觀
```
[目錄] [設定] [☰] [朗讀]    ← 手機版（僅圖標）
[目錄] [設定] [更多] [朗讀]  ← 桌面版（圖標+文字）
```

### 下拉選單內容
```
┌─────────────────────────┐
│  🏠 首頁                │
│  📚 書庫                │
│  👤 我的                │
│  📖 書籍詳情頁          │
└─────────────────────────┘
```

### 行為規範
1. 點擊「更多」按鈑 → 彈出下拉面板（使用現有 sheet 系統）
2. 點擊選單選項 → 跳轉對應頁面並關閉面板
3. 點擊遮罩層或再次點擊「更多」→ 關閉面板

## 技術實現

### 修改文件
- `src/pages/book/[novel]/[chapter].astro`

### 新增內容
1. 新增「更多」按鈑在底部工具列（位於設定和朗讀之間）
2. 新增 `moreSheet` 面板，包含四個導航選項
3. 使用現有的 `.sheet-overlay` 和 `.sheet-content` 樣式系統

### 現有程式碼複用
- 沿用 `openSheet()` / `closeSheet()` 函數
- 沿用 `.sheet-overlay` CSS 樣式
- 沿用 `.toc-item` 樣式或創建類似的 `.nav-item` 樣式

2. ✅ 桌面版顯示「更多」文字+圖標

5. ✅ 點擊選項可正確跳轉到對應頁面
6. ✅ 點擊遮罩層或再次點擊「更多」可關閉面板
